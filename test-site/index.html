welcome to my cashew site

this is being served through:

  • cashew node (localhost:8080) - stores the content
  • perl CGI gateway (localhost:8081) - serves it to your browser
  • content-addressed storage (BLAKE3 hash)

if you're seeing this, the framework works.

how this works:

  1. website files uploaded to cashew node
  2. node hashes them with BLAKE3
  3. hash becomes the content ID
  4. gateway requests content by hash
  5. node serves it, gateway renders it
  6. you see this page

made with cashew - it compiles and probably works
